# My WebGIS Map - Lab 6 Final
 
## Description
This web map shows points for Los Angales, Houston, and Chicago w/various details.
It was built using Leaflet.js as part of GIS 5220/420 Lab 6.
 
## Data Source
The data used in this map comes from the US Census Bureau.
 
## Features
- Interactive basemap with zoom and pan
- GeoJSON point markers for US cities (Houston, LA, Chicago).
- Clickable popups showing city name and population
- Scale bar showing metric and imperial units
 
## Technologies Used
- HTML5, CSS3, JavaScript (all in one index.html file)
- Leaflet.js 1.9.4
- OpenStreetMap tiles
 
## Live Deployments
